1. Open illustrator
  2. Open a letter sized document
  3. Select the pen tool
  4. Make a rectangle that fills the entire lower half of the page
  5. Make that rectangle red
  6. Make a new layer
  7. Select the pen tool again
  8. Create a new rectangle inside of the last one that stretches from the bottom of the red rectangle to about the vertical center of the red rectangle.
  9. Make sure that rectangle is taller than the width and is evenly centered in the red rectangle with the bottom touching the base of the document
  10. Color the new rectangle black
  11. Create a new layer
  12. Select the pen tool one last time
  13. Make an upwards pointing triangle that stretches from the top of the red rectangle to the top of the document
  14. Make sure that the bottom of the triangle is connected to the top of the red rectangle and that the top point is connected to the top of the document
  15. Color the triangle brown
  16. Make sure that the triangle is in the top half of the document
  17. Make sure that the red rectangle is in the lower half of the document
  18. Make sure that the black rectangle creates a door like shape in the bottom of the red rectangle
  19. Congratulations if your drawing looks like a house then you were successful
